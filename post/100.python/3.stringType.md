## 문자열 만드는 방법

1. 큰따옴표로 둘러싸기
   - "문자열"
2. 작은따옴표로 둘러싸기
   - '문자열'
3. 큰따옴표 3개로 둘러싸기
   - """문자열"""
4. 작은따옴표 3개로 둘러싸기
   - '''문자열'''

- 3,4번같은경우, 멀티라인이 가능하다
- 1,2번같은경우,멀티라인 사용시, \n을 끼워 넣어야한다

## 문자열 안에서 작은따옴표, 큰따옴표

### 문자열 안에 작은따옴표, 큰따옴표 포함시킬려면, 백슬래시(`\`)를 사용한다

>>> say = "\"python is very easy\" he says."

## 문자열 슬라이싱

`a[0:3] === 0 <= a < 3`

```Python
>>> a="abcde"
>>> a[-1] # 끝에서 첫번째
e
# 왜 -0이 아닌걸까? -0은 0과 같아서 -1이 끝에서 첫번째임.
>>> a[-2]
d
>>> a[0:2]
ab # a[시작번호 : 끝번호]
>>> a[0:]
abcde #끝번호 생략시, 문자열 끝까지.
>>> a[:2]
ab #시작번호 생략시, 문자열 처음부터.
>>> a[:]
abcde #문자열 처음~ 끝.
```

## 문자열 변경 및 수정

### 잘못된 케이스

```Python
>>> a = "Pithon"
>>> a[1]
'i'
>>> a[1] = 'y'
```

- 문자열의 요솟값은 바꿀수 있는 값이 아니기 때문이다.(immutable한 자료형.)

### 올바른 예

```Python
>>> a = "Pithon"
>>> a[:1]
'P'
>>> a[2:]
'thon'
>>> a[:1] + 'y' + a[2:]
'Python'
```

- 새로운 문자열을 만들어야 한다

## 문자열 포맷팅

### 숫자일때 %d, 문자열일떄 %s

```Python
>>> "I eat %d apples." % 3
'I eat 3 apples.'
>>> "I eat %s apples." % "five"
'I eat five apples.

```

### 2개 이상일때.

```Python
>>> number = 10
>>> day = "three"
>>> "I ate %d apples. so I was sick for %s days." % (number, day)
'I ate 10 apples. so I was sick for three days.'
```

### 문자열 포맷 코드

| 코드 | 설명                     |
| ---- | ------------------------ |
| %s   | 문자열(String)           |
| %c   | 문자 1개(character)      |
| %d   | 정수(Integer)            |
| %f   | 부동소수(floating-point) |
| %o   | 8진수                    |
| %x   | 16진수                   |

- 문자열 내에서 %를 사용하려면, %%로 입력한다.

### 문자열 포맷코드 공백, 정렬

```Python
>>> "%10s" % "hi"
'        hi' #문자열 길이가 10, 우측정렬,나머지 공백.
>>> "%-10sjane." % 'hi'
'hi        jane.' # 왼쪽정렬은 %-10s
>>> "%0.4f" % 3.42134234
'3.4213' # 소숫점 자릿수  4자리까지만,
>>> "%10.4f" % 3.42134234
'    3.4213' # 전체길이는 10인데, 소숫점자릿수는 4자리,우측정렬.
```

## format 함수를 이용한 포맷팅

```Python
#숫자
>>> "I eat {0} apples".format(3)
'I eat 3 apples'
#문자열
>>> "I eat {0} apples".format("five")
'I eat five apples'
# 2개이상
>>> number = 10
>>> day = "three"
>>> "I ate {0} apples. so I was sick for {1} days.".format(number, day)
'I ate 10 apples. so I was sick for three days.'
#이름으로 넣기
>>> "I ate {number} apples. so I was sick for {day} days.".format(number=10, day=3)
'I ate 10 apples. so I was sick for 3 days.'
#인덱스 혼용
>>> "I ate {0} apples. so I was sick for {day} days.".format(10, day=3)
'I ate 10 apples. so I was sick for 3 days.'
# :<10 표현식으로 왼쪽정렬, 전체길이 10,
>>> "{0:<10}".format("hi")
'hi        '
# :>10 표현식으로 우츨정렬,전체길이 10
>>> "{0:>10}".format("hi")
'        hi'
# :^ 표현식으로 가운데 정렬,전체길이 10
>>> "{0:^10}".format("hi")
'    hi    '
# 공백채우기, :(콜론) 과 >(정렬) 사이에, 문자열을 삽입.
>>> "{0:=^10}".format("hi")
'====hi===='
>>> "{0:!<10}".format("hi")
'hi!!!!!!!!'
## {, } 문자 사용하기
>>> "{{  }}".format()
'{  }'
```

## f 문자열 포맷팅

문자열 앞에, f 접두사 붙이면 format 기능을 사용할수있다

```Python
##변수사용
>>> name = '홍길동'
>>> age = 30
>>> f'나의 이름은 {name}입니다. 나이는 {age}입니다.'
'나의 이름은 홍길동입니다. 나이는 30입니다.'
##정렬
>>> f'{"hi":<10}'  # 왼쪽 정렬
'hi        '
>>> f'{"hi":>10}'  # 오른쪽 정렬
'        hi'
>>> f'{"hi":^10}'  # 가운데 정렬
'    hi    '
## 소숫점
>>> y = 3.42134234
>>> f'{y:0.4f}'  # 소수점 4자리까지만 표현
'3.4213'
>>> f'{y:10.4f}'  # 소수점 4자리까지 표현하고 총 자리수를 10으로 맞춤
'    3.4213'
```

## 문자열 관련 내장함수들

### count

```Python
>>> a = "hobby"
>>> a.count('b') # b의 개수
2
```

### find

- 청므으로 찾은 인덱스 반환, 못찾으면 -1.

```Python
>>> a = "abcde"
>>> a.find('b') # 처음으로 찾은 b의 인덱스
1
>>> a.find('k') #못 찾으면 -1
-1
```

### index

- 처음으로 찾은 인덱스를 반환. 못찾으면 error.

```Python
>>> a = "abcde"
>>> a.index('e')
4
>>> a.index('k') # 못찾으면 error.
Traceback (most recent call last):
File "<stdin>", line 1, in <module>
ValueError: substring not found
```

### join

- 문자열 삽입

```Python
>>> ",".join('abcd')
'a,b,c,d'
```

### upper

- 대문자로 바꾸기

```Python
>>> a = "hi"
>>> a.upper()
'HI'
```

### lower

- 소문자로 바꾸기

```Python
>>> a = "HI"
>>> a.lower()
'hi'
```

### lstrip

- 왼쪽 공백 지우기

```Python
>>> a = " hi "
>>> a.lstrip()
'hi '
```

### rstrip

- 오른쪽 공백 지우기

```Python
>>> a= " hi "
>>> a.rstrip()
' hi'
```

### strip

- 양쪽 공백 지우기

```Python
>>> a = " hi "
>>> a.strip()
'hi'
```

### replace

- replace(바뀌게 될 문자열, 바꿀 문자열)
- 문자열 바꾸기.

```Python
>>> a = "Life is too short"
>>> a.replace("Life", "Your leg")
'Your leg is too short'
```

### split

- 문자열 나눠서 배열로 반환.

```Python
>>> a = "Life is too short"
>>> a.split()
['Life', 'is', 'too', 'short']
>>> b = "a:b:c:d"
>>> b.split(':')
['a', 'b', 'c', 'd']
```
