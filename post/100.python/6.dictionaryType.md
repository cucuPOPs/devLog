- key, value를 한쌍으로 갖는 자료형이다.
- key 는 변하지 않는값 사용. ex) 튜플.
- value에는 변하는값, 변하지않는값 모두가능.

## 딕셔너리 기본형태

```Python
{Key1:Value1, Key2:Value2, Key3:Value3, ...}
```

## 딕셔너리 추가,삭제.

### 추가

```Python

>>> a = {1: 'a'}
>>> a[2] = 'b' # 추가.
>>> a
{1: 'a', 2: 'b'} # 추가된것을 확인.
>>> a['name'] = 'pey' #추가.
>>> a
{1: 'a', 2: 'b', 'name': 'pey'} #추가되엇음.
```

### 삭제

- `del a[key]`처럼 입력하면 지정한 Key에 해당하는 {key : value} 쌍이 삭제된다.

```Python
>>> del a[1]
>>> a
{2: 'b', 'name': 'pey', 3: [1, 2, 3]}
```

## 딕셔너리 추가시 주의

- 동일한 Key가 2개 존재할 경우, 마지막에 입력된게 최종적으로 남음.

```Python
>>> a = {1:'a', 1:'b'}
>>> a
{1: 'b'}
```

## 딕셔너리 관련 함수들

### keys()

- 딕셔너리 key만을 모아서, dict_keys객체를 반환한다.

```Python
>>> a = {'name': 'pey', 'phone': '0119993323', 'birth': '1118'}
>>> a.keys()
dict_keys(['name', 'phone', 'birth'])
# a.keys()는 딕셔너리 a의 Key만을 모아서 dict_keys 객체를 돌려준다.
```

### values()

```Python
>>> a = {'name': 'pey', 'phone': '0119993323', 'birth': '1118'}
a.values()
dict_values(['pey', '0119993323', '1118'])
```

### items()

- key, value 쌍을 튜플로 묶은값을 dict_items 객체로 돌려준다.

```Python
>>> a = {'name': 'pey', 'phone': '0119993323', 'birth': '1118'}
>>> a.items()
dict_items([('name', 'pey'), ('phone', '0119993323'), ('birth', '1118')])
```

### clear()

- key value 쌍 모두 지우기

```Python
>>> a.clear()
>>> a
{}
```

### get()

- get(x) 함수는 x라는 Key에 대응되는 Value를 돌려준다.

```Python
>>> a = {'name':'pey', 'phone':'0119993323', 'birth': '1118'}
>>> a.get('name')
'pey'
>>> a.get('phone')
'0119993323'
>>> print(a.get('nokey'))
None
# 해당하는게 없을때, None를 돌려줌.
# None: "거짓"이라는뜻.
>>> print(a['nokey'])
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
KeyError: 'nokey'
# get사용안할시, 이렇게 에러.

>>> a.get('foo', 'bar')
'bar'
# 디폴트값 지정 가능.
```

### in()

```Python
>>> a = {'name':'pey', 'phone':'0119993323', 'birth': '1118'}
>>> 'name' in a
True
>>> 'email' in a
False
```
