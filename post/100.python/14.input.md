## input

- 입력되는 모든것을 문자열로 취급한다

```Python
>>> a = input()
Life is too short, you need python
>>> a
'Life is too short, you need python'
>>>
```

```Python
>>> number = input("숫자를 입력하세요: ")

```

## 파일 입출력

- `파일 객체 = open(파일 이름, 파일 열기 모드)`
  | 파일 열기 모드 |설명 |
  | -------------- | ---------------------------------------------------------- |
  | r | 읽기모드 - 파일을 읽기만 할 때 사용 |
  | w | 쓰기모드 - 파일에 내용을 쓸 때 사용 |
  | a | 추가모드 - 파일의 마지막에 새로운 내용을 추가 시킬 때 사용 |

```Python
f = open("C:/doit/새파일.txt", 'w') # 이렇게 경로를 다 써줘도 된다.
f.close()

f = open("새파일.txt", 'w') # 실행한 경로에, 쓰기모드로, 새파일.txt 오픈.
f.close() #  쓰기모드로 열었던 파일을 닫지 않고 다시 사용하려고 하면 오류가 발생하기 때문에, 닫아주는것이 좋다.
```

### 파일 쓰기

```Python
f = open("C:/doit/새파일.txt", 'w')
for i in range(1, 11):
    data = "%d번째 줄입니다.\n" % i
    f.write(data)
f.close()
# 파일을 열어보면, 1번째 줄입니다 ~ 10번째 줄입니다 텍스트가 입력되어있음.
```

### 파일 읽기

```Python
#readline() 으로 읽는방법
f = open("C:/doit/새파일.txt", 'r')
line = f.readline() # 더이상 읽을 줄이 없을경우, 빈문자열("")을 반환함.
print(line)
f.close()

# readlines()로 읽는방법
f = open("C:/doit/새파일.txt", 'r')
lines = f.readlines() #리스트로 반환함.
for line in lines:
    print(line)
f.close()
#["1번째 줄입니다", "2번쨰 줄입니다", ..., "10번째 줄입니다"]


# read()로 읽는방법
f = open("C:/doit/새파일.txt", 'r')
data = f.read()
print(data)
f.close()
# 파일의 내용전체를 문자열로 반환한다.
```
