```Python
while <조건문>:

    ...
```

## continue

```Python
>>> a = 0
>>> while a < 10:
...     a = a + 1
...     if a % 2 == 0: continue
...     print(a)
...
1
3
5
7
9
```

## break

```Python
>>> a = 0
>>> while a < 10:
...     a = a + 1
...     if a % 2 == 0: break
...     print(a)
...
1
```
