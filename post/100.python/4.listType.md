- 리스트를 만들 때는 위에서 보는 것과 같이 대괄호([ ])로 감싸 주고 각 요솟값은 쉼표(,)로 구분해 준다.

```Python
리스트명 = [요소1, 요소2, 요소3, ...]
```

- ※ 비어 있는 리스트는 `a = list()`로 생성할 수도 있다.

## 리스트의 슬라이싱

`seq[시작:끝:스텝]`

```Python
>>> a = [1, 2, 3, 4, 5]
>>> a[0:2]
[1, 2] # 배열을 반환하네?
```

```Python
>>> a = "12345"
>>> a[0:2]
'12' # 문자열이네?
```

- 아하, 슬라이싱은 단어뜻 그대로 나눈다는 의미.

## 리스트 연산

### 리스트 더하기

```Python
>>> a = [1, 2, 3]
>>> b = [4, 5, 6]
>>> a + b
[1, 2, 3, 4, 5, 6]
```

### 리스트 반복

```Python
>>> a = [1, 2, 3]
>>> a * 3
[1, 2, 3, 1, 2, 3, 1, 2, 3]
```

### 리스트 연산시 TypeError

```Python
>>> a = [1, 2, 3]
>>> a[2] + "hi"
```

```Python
>>> a = [1, 2, 3]
>>> str(a[2]) + "hi" # int를 str로 바꿔줌 str은 내장함수.
```

## 리스트 수정,삭제

### 리스트 수정

```Python
>>> a = [1, 2, 3]
>>> a[2] = 4
>>> a
[1, 2, 4]
```

### 리스트 삭제

```Python
>>> a = [1, 2, 3]
>>> del a[1] # del 함수는 파이썬 내장함수.
>>> a
[1, 3]
```

```Python
>>> a = [1, 2, 3, 4, 5]
>>> del a[2:] # 이렇게도 삭제가능.
>>> a
[1, 2]
```

## 리스트 관련 함수(수정,삭제)

- append, sort, reverse, index,

### append

```Python
#리스트 끝에 추가.
>>> a.append([5,6])
>>> a
[1, 2, 3, 4, [5, 6]]
```

### sort

```Python
# 숫자 정렬
a = [1, 4, 3, 2]
a.sort()
print(a)  # [1, 2, 3, 4]
a.sort(reverse=True)
print(a)  # [4,3,2,1]

```

```Python
#알파벳 정렬
>>> a = ['a', 'c', 'b']
>>> a.sort()
>>> a
['a', 'b', 'c']
```

### reverse

```Python
# 리스트를 역순으로 뒤집기
>>> a = ['a', 'c', 'b']
>>> a.reverse()
>>> a
['b', 'c', 'a']
```

### index

- 못찾을시, 타입에러.

```Python
>>> a = [1,2,3]
>>> a.index(3)
2
>>> a.index(1)
0
```

### insert

- insert(a, b)는 리스트의 a번째 위치에 b를 삽입하는 함수이다

```Python
>>> a = [1, 2, 3]
>>> a.insert(0, 4)
>>> a
[4, 1, 2, 3]
```

### remove

- 첫번째로 찾은 요소 1개만 삭제됨.

```Python
>>> a = [1, 2, 3, 1, 2, 3]
>>> a.remove(3)
>>> a
[1, 2, 1, 2, 3] # 맨뒤의 3은 제거되지않았음.
```

### pop

```Python
>>> a = [1,2,3]
>>> b = a.pop()
>>>> print("a는",a)
[1,2]
>>>> print("b는",b)
3
```

```Python
>>> a = [1,2,3]
>>> a.pop(1)
2
>>> a
[1, 3]
```

### count

```Python
>>> a = [1,2,3,1]
>>> a.count(1) # 숫자1이 몇개있는지 체크.
2
```

### extend

- extend(x)에서 x에는 리스트만 올 수 있으며 원래의 a 리스트에 x 리스트를 더하게 된다.

```Python
>>> a = [1,2,3]
>>> a.extend([4,5])
>>> a
[1, 2, 3, 4, 5]
>>> b = [6, 7]
>>> a.extend(b)
>>> a
[1, 2, 3, 4, 5, 6, 7]
```

- a.extend([4, 5])는 a += [4, 5]와 동일하다.
