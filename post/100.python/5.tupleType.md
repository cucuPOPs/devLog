## 리스트와 다른점

- 리스트는 [ ]으로 둘러싸지만 튜플은 ( )으로 둘러싼다.
- 리스트는 그 값의 생성, 삭제, 수정이 가능하지만 튜플은 그 값을 바꿀 수 없다.
  - (요소수정,요소삭제 모두 불가능)
- 튜플의 요소에 del 함수를 사용하면 error가 나온다.

```Python
>>> t1 = ()
>>> t2 = (1,) # 1개요소만을 가질때, 콤마를 반드시 붙여야 한다.
>>> t3 = (1, 2, 3)
>>> t4 = 1, 2, 3
>>> t5 = ('a', 'b', ('ab', 'cd'))
```

## 튜플 요소값을 삭제하려 할때

```Python
>>> t1 = (1, 2, 'a', 'b')
>>> del t1[0]
```
