- 파이썬은 인덴트(들여쓰기) 가 중요하다.

```Python
if 조건문:
    수행할 문장1
    수행할 문장2
    ...
else:
    수행할 문장A
    수행할 문장B
    ...
```

## and, or, not

| 연산자  | 설명                                |
| ------- | ----------------------------------- |
| x or y  | x와 y 둘중에 하나만 참이어도 참이다 |
| x and y | x와 y 모두 참이어야 참이다          |
| not x   | x가 거짓이면 참이다                 |

## x in s, not in s

- foreach 같은게 아니라, contains()같은 느낌.
  | in | not in |
  | ----------- | --------------- |
  | x in 리스트 | x not in 리스트 |
  | x in 튜플 | x not in 튜플 |
  | x in 문자열 | x not in 문자열 |

```Python
>>> 1 in [1, 2, 3]
True
>>> 1 not in [1, 2, 3]
False
```

## pass

- 단순히 실행할 코드가 없다는 의미.

```Python
>>> pocket = ['paper', 'money', 'cellphone']
>>> if 'money' in pocket:
...     pass
... else:
...     print("카드를 꺼내라")
...
```

## 중첩 if문

```Python
 if <조건문>:
    <수행할 문장1>
    <수행할 문장2>
    ...
elif <조건문>:
    <수행할 문장1>
    <수행할 문장2>
    ...
elif <조건문>:
    <수행할 문장1>
    <수행할 문장2>
    ...
...
else:
   <수행할 문장1>
   <수행할 문장2>
   ...
```
