- `call by reference` vs `call by value`

## 리스트의 복사

```Python
a = [1, 2, 3]
b = a  # 같은 주소값
b = a[:]  # 새로운 리스트 할당
b = list(a)  # 새로운 리스트 할당
```

## set의 복사

```Python
set1 = {"1", "2", "3"}
set2 = set1.copy()
set1.remove("1")
set1.discard("2")
print(set1)  # {'3'}
print(set2)  # {'1','2','3'}
```

## dict 의 복사

```Python
dict1 = {"red": "apple"}
dict2 = dict1.copy()
dict1.update({"red": "Strawberry"})
print(dict1) #{'red': 'Strawberry'}
print(dict2) #{'red': 'apple'}
```

## Swap

```Python
a,b= 3,5
a, b = b, a # tupple, swap
print(a) # 5
print(b) # 3
```
