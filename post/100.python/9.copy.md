date: "2021-03-25"
---

- `얉은복사` vs `깊은복사`
- `copy by reference` vs `copy by value`

```Python
>>> a = [1,2,3]
>>> b = a
# a,b모두 같은 메모리를 참조한다
>>> a[1] = 4 #a를 수정할시,
>>> a
[1, 4, 3] #a도 변하고,
>>> b
[1, 4, 3]# b도 변한다.
# a변수 메모리와, b변수 메모리가 같음.
```

```Python
>>> a = [1, 2, 3]
>>> b = a[:] # 새로운 메모리에 할당됨
>>> a[1] = 4
>>> a
[1, 4, 3]
>>> b
[1, 2, 3]
# a메모리와 b메모리가 다름.
```

```Python
#Swap
>>> a = 3
>>> b = 5
>>> a, b = b, a # 튜플은 괄호 생략 가능.
>>> a
5
>>> b
3
```
