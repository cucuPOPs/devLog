```Python
>>> class className:
...     pass
...
>>>
```

## 계산기 클래스 만들기

```Python
>>> class FourCal:
...     def setdata(self, first, second): # self : 자기자신의 객체를 뜻함.
...         self.first = first
...         self.second = second
...     def add(self):
...         result = self.first + self.second
...         return result
...     def mul(self):
...         result = self.first * self.second
...         return result
...     def sub(self):
...         result = self.first - self.second
...         return result
...     def div(self):
...         result = self.first / self.second
...         return result
...
>>>
```

## 계산기 클래스 사용.

- a객체, b객체 따로 저장공간을 가진다.

```Python
>>> a = FourCal()
>>> b = FourCal()
>>> a.setdata(4, 2)
>>> b.setdata(3, 8)
>>> a.add()
6
>>> a.mul()
8
>>> a.sub()
2
>>> a.div()
2
>>> b.add()
11
>>> b.mul()
24
>>> b.sub()
-5
>>> b.div()
0.375
```

## 생성자

```Python
>>> class FourCal:
...     def __init__(self, first, second): ## 언더바 2개 __init__
...         self.first = first
...         self.second = second
...     def setdata(self, first, second):
...         self.first = first
...         self.second = second
...     def add(self):
...         result = self.first + self.second
...         return result
.
.
.
```

```Python
>>> a = FourCal(4, 2) # 이렇게 객체생성시, 값을 주어야함.
>>> a.add()
6
>>> a.div()
2.0
```

## 클래스의 상속

- 클래스를 상속하기 위해서는 다음처럼 클래스 이름두 ㅣ괄호안에 상속할 클래스 이름을 넣어주면 된다.

```Python
>>> class MoreFourCal(FourCal):
...     pass
...
>>>
```

### 기존계산기 클래스를 상속하여, pow 기능추가.

```Python
>>> class MoreFourCal(FourCal):
...     def pow(self):
...         result = self.first ** self.second
...         return result
...
>>>
```

## 클래스 변수

- 클래스 인스턴스에 대한 변수가 아니라, 전역변수.

```Python
>>> class Family:
...     lastname = "김"
...
# 클래스를 통해,
>>> print(Family.lastname)
김
# 인스턴스를 통해,
>>> a = Family()
>>> b = Family()
>>> print(a.lastname)
김
>>> print(b.lastname)
김
```
