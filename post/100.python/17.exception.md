## 기본형태

```Python
try:
    ...
except:
    ...
else:  # 오류가 없을 경우에만 수행된다.
...
```

## 샘플

```Python
try:
    4 / 0
except ZeroDivisionError as e:
    print(e)
```

## 에러 강제발생

```Python
class Bird:
    def fly(self):
        raise NotImplementedError
```
