## 메타문자

`. ^ $ * + ? { } [ ] \ | ( )`

- 메타 문자란 원래 그 문자가 가진 뜻이 아닌 특별한 용도로 사용하는 문자를 말한다.

## 문자클래스 [ ]

### "[] 사이의 문자들과 매치" 라는 의미.

- `[abc]` : "a, b, c 중 한 개의 문자와 매치"
- `[0-5]` : `[012345]` 의 줄임표현.
- `[a-zA-Z]` : 알파벳 모두
- `[0-9]` : 숫자
- `[^0-9]` : 숫자가 아닌것. ^은 not반대라는 의미.
- `\d` : `[0-9]` 와 동일.
- `\D` : `[^0-9]` 와 동일.
- `\s` : `[ \t\n\r\f\v]`와 동일. (whitespace 문자와 매치.)
- `\S` : `[^ \t\n\r\f\v]` 와 동일.
- `\w` : `[a-zA-Z0-9_]` 와 동일. 문자+ 숫자와 매치.
- `\W` : `[^a-zA-Z0-9_]`와 동일. 문자+숫자가 아닌것과 매치.

## Dot(.)

### 줄바꿈(\n) 제외한 모든 문자와 매치.

- `a.b` : "a"+ 모든문자(1개이상) +"b". "abc"는 매치되지않음 . "aaab"는 매치됨.
- `a[.]b` : `a.b`와 매치

## 반복(\*)

- \*(별표) 바로앞에 있는 문자a가 **0부터** 무한대로 반복될수있다는 의미.

| 정규식 | 문자열 | Match 여부 | 설명                                    |
| ------ | ------ | ---------- | --------------------------------------- |
| ca\*t  | ct     | Yes        | "a"가 0번 반복되어 매치!!!!!!!!!!       |
| ca\*t  | cat    | Yes        | "a"가 0번 이상 반복되어 매치 (1번 반복) |
| ca\*t  | caaat  | Yes        | "a"가 0번 이상 반복되어 매치 (3번 반복) |

## 반복(+)

- 최소1번이상 반복일때 사용.

| 정규식 | 문자열 | Match 여부 | 설명                                    |
| ------ | ------ | ---------- | --------------------------------------- |
| ca+t   | ct     | No         | "a"가 0번 반복되어 매치되지 않음        |
| ca+t   | cat    | Yes        | "a"가 1번 이상 반복되어 매치 (1번 반복) |
| ca+t   | caaat  | Yes        | "a"가 1번 이상 반복되어 매치 (3번 반복) |

## 반복( {m,n}, ?)

- {3,} : 반복횟수가 3이상인 경우.
- {,3} : 반복횟수가 3 이하인 경우.
- {0,무한대} : 기본값.
- {1,} : + 와 동일
- {0,} : \* 와 동일

| 정규식 | 문자열 | Match 여부 | 설명                               |
| ------ | ------ | ---------- | ---------------------------------- |
| ca{2}t | cat    | No         | "a"가 1번만 반복되어 매치되지 않음 |
| ca{2}t | caat   | Yes        | "a"가 2번 반복되어 매치            |

### ?

- {0,1} 과 같다.

| 정규식 | 문자열 | Match 여부 | 설명                    |
| ------ | ------ | ---------- | ----------------------- |
| ab?c   | abc    | Yes        | "b"가 1번 사용되어 매치 |
| ab?c   | ac     | Yes        | "b"가 0번 사용되어 매치 |

## 가급적 {m,n} 형태보다, \*,+,? 사용권장.
