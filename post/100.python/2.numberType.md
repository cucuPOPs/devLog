## 정수형

```python
>>> a = -178
>>> a = 0
```

## 실수형

```python
>>> a = -3.45
>>> a = 4.24E10  # 4.24E10 = 4.24 x 10^10
>>> a = 4.24E-10  # 4.24E-10 = 4.24 x 10^-10
```

## 8진수(Octal) 16진수(Hexadecimal)

```python
#8진수, 0o 또는 0O로 시작하면 된다.
>>> a = 0o177 # 숫자0, 소문자알파벳o
>>> a = 0O177 #숫자0,대문자알파벳O
```

```python
#16진수 0x로 시작하면 된다.
>>> a = 0x8ff
>>> a = 0xABC
```

## 사칙연산

`+`, `-`, `*`, `/` 은 계산기 기능과 같다

## \*\* 연산자 ( 거듭제곱 )

```Python
>>> a = 3
>>> a = 4
>>> a ** b # x의 y제곱, 3^4
81
```

## % 연산자

```Python
# 나머지를 구하는 연산자
>>> 7 % 3 #7을 3으로 나누면 나머지는 1
1
```

## // 연산자

```Python
# 몫을 구하는 연산자
>>> 7 / 4  #나누기 연산을 하면, 1.75
1.75
>>> 7 // 4 # //연산을 하면 1
1
```
