## all

- all(x)는 반복 가능한(iterable) 자료형 x를 입력 인수로 받으며 이 x의 요소가 모두 참이면 True, 거짓이 하나라도 있으면 False를 돌려준다.

```Python
>>> all([1, 2, 3])
True
>>> all([1, 2, 3, 0])
False
>>> all([])
True
```

## any

- any(x)는 반복 가능한(iterable) 자료형 x를 입력 인수로 받으며 이 x의 요소 중 하나라도 참이 있으면 True를 돌려주고, x가 모두 거짓일 때에만 False를 돌려준다. all(x)의 반대이다.

```Python
>>> any([])
False
>>> any([0, ""])
False
>>> any([1, 2, 3, 0])
True
```

## chr

- chr(i)는 아스키(ASCII) 코드 값을 입력받아 그 코드에 해당하는 문자를 출력하는 함수이다.

```Python
>>> chr(97)
'a'
>>> chr(48)
'0'
```

## sorted

- sorted(iterable) 함수는 입력값을 정렬한 후 그 결과를 리스트로 돌려주는 함수이다.
- 리스트 자료형에도 sort 함수가 있다.

```Python
>>> sorted([3, 1, 2])
[1, 2, 3]
>>> sorted(['a', 'c', 'b'])
['a', 'b', 'c']
>>> sorted("zero")
['e', 'o', 'r', 'z']
>>> sorted((3, 2, 1))
[1, 2, 3]

# 리스트 sort 와 차이.
# 리스트는 원본을 정렬함.
# sorted는 원본을 변경하지 않음.

b = [3, 1, 2]
print(b.sort()) #None.
print(b)#[1,2,3]

a=[3,1,2]
print(sorted(a)) #[1,2,3]
print(a) # [3,1,2]
```

## zip

- 자료형을 묶어주는 함수

```Python
>>> list(zip([1, 2, 3], [4, 5, 6]))
[(1, 4), (2, 5), (3, 6)]
>>> list(zip([1, 2, 3], [4, 5, 6], [7, 8, 9]))
[(1, 4, 7), (2, 5, 8), (3, 6, 9)]
>>> list(zip("abc", "def"))
[('a', 'd'), ('b', 'e'), ('c', 'f')]
```
