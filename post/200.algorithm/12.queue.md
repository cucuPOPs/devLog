선입선출 FIFO 구조.

## ADT

enqueue : 큐 뒤쪽에 항목을 삽입
dequeue : 큐 앞쪽의 항목을 반환하고, 제거
peek/front : 큐 앞쪽의 항목을 조회
isEmpty : 큐가 비어있는지 확인
size : 큐의 크기를 확인

## 구현

```Python
class Queue(object):
    def __init__(self):
        self.items = []

    def isEmpty(self):
        return not bool(self.items)

    def enqueue(self, item):
        self.items.insert(0, item)

    def dequeue(self):
        value = self.items.pop()
        if value is not None:
            return value
        else:
            print("Queue is empty.")

    def size(self):
        return len(self.items)

    def peek(self):
        if self.items:
            return self.items[-1]
        else:
            print("Queue is empty.")

    def front(self):
        return self.peek()

    def __repr__(self):
        return repr(self.items)


q = Queue()
print("isEmpty:", q.isEmpty())
for i in range(10):
    q.enqueue(i)
print("q:", q)  # [9, 8, 7, 6, 5, 4, 3, 2, 1, 0]
print("size:", q.size())  # 10
print("peek:", q.peek())  # 0
print("front:", q.front())  # 0
print("dequeue:", q.dequeue())  # 0
print("peek:", q.peek())  # 1
print("isEmpty:", q.isEmpty())  # False
print("q:", q)  # [9, 8, 7, 6, 5, 4, 3, 2, 1]

```
