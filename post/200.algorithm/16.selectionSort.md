## 선택정렬이란?

가장 작은(큰) 항목을 찾아서 1번째와 위치를 바꾼다.
나머지 항목들에서 가장 작은(큰) 항목을 찾아서 2번째와 위치를 바꾼다.
리스트 끝에 도달할때 까지 이 과정을 반복한다

## 소스코드

```Python
def selection_sort(seq):
    length = len(seq)
    for i in range(0, length - 1, 1):
        min_j = i
        for j in range(i + 1, length, 1):
            if seq[min_j] > seq[j]:
                min_j = j
        seq[i], seq[min_j] = seq[min_j], seq[i]
    return seq


l1 = [11, 3, 28, 43, 9, 4]
selection_sort(l1)
print(l1) #[3, 4, 9, 11, 28, 43]
```
