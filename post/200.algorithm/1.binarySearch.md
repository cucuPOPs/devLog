## 소스코드

- list는 정렬된 상태에서 시작.
- 탐색할 값을 넣으면, 찾은 인덱스를 반환. 못찾으면 None을 반환.

```Python
def binarySearch(list, targetValue):
    low = 0
    high = len(list) - 1
    while low <= high:
        mid = (low + high) // 2 ## 몫을 구하는 연산자
        guess = list[mid]
        if guess == targetValue:
            return mid
        if guess > targetValue:
            high = mid - 1
        else:
            low = mid + 1
        return None

def test_bs():
    myList=[10,20,30,40,50]
    assert(binarySearch(myList,30) == 2)
    assert(binarySearch(myList,100) == None)
    print("테스트통과!")

```
