## 버블정렬이란?

인접한 두 항목을 비교하여 정렬하는 방식
수면위로 거품이 올라오는듯한 모습을 보이기에 붙은 이름

```Python
def bubble_sort(seq):
    length = len(seq) - 1
    for num in range(length, 0, -1):
        for i in range(0, num, 1):
            if seq[i] > seq[i + 1]:
                seq[i], seq[i + 1] = seq[i + 1], seq[i]
    return seq
l1 = [11, 3, 28, 43, 9, 4]
bubble_sort(l1)
print(l1) #[3, 4, 9, 11, 28, 43]
```
