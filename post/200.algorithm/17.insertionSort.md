## 삽입정렬이란?

정렬도니 부분에, 정렬되지않은 다음항목을 반복적을 삽입하는 방식이다
미리 정렬된 리스트에 새 항목을 추가할때 좋다

## 소스코드

```Python
def insertion_sort(arr):
    for end in range(1, len(arr)):
        for i in range(end, 0, -1):
            if arr[i - 1] > arr[i]:
                arr[i - 1], arr[i] = arr[i], arr[i - 1]

l1 = [11, 3, 28, 43, 9, 4]
insertion_sort(l1)
print(l1)
```
