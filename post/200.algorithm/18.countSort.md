## 카운트 정렬이란

작은범위의 정수를 정렬할때 유용하다

```Python
# counting sort 구현
def counting_sort(array, max):

    counting_array = [0] * max
    output_array = [0] * len(array)

    # 원소의 빈도수 담기
    for i in array:
        counting_array[i] += 1
    # counting array 값 누적시키기
    for i in range(1, max):
        counting_array[i] += counting_array[i - 1]

    # output array에 정렬하기(counting array를 참조)
    for i in range(len(array)):
        output_array[counting_array[array[i]] - 1] = array[i]
        counting_array[array[i]] -= 1
    return output_array


l1 = [3, 5, 2, 6, 8, 1, 0, 3, 5, 6, 2, 5, 4, 1, 5, 3]
print(counting_sort(l1, 100))  #
```
