## 개념적 기본형태

```Python
def prime(number):
    num = abs(number)
    if num < 4:
        return True
    for x in range(2, num):
        if num % x == 0:
            return False
    return True
```

## 제곱근 이용

```Python
def prime_sqrt(number):
    num = abs(number)
    if num < 4:
        return True
    for x in range(2, int(math.sqrt(num)) + 1):
        if number % x == 0:
            return False
    return True
```

## 에라토스테네스의 체

- 여러 소수를 구할때 용이함

```Python
def prime_era(maxRanage):
    a = [False, False] + [True] * (maxRanage - 1)
    primes = []

    for i in range(2, maxRanage + 1):
        if a[i]:
            primes.append(i)
            for j in range(2 * i, maxRanage + 1, i):
                a[j] = False
    return primes
```

## 페르마의 소정리

- 큰 수의 범위에서, 확률적으로 소수판별하기때문에 불완전.

```Python
def prime_fermat(number):
    if number <= 102:
        for a in range(2, number):
            if pow(a, number - 1, number) != 1:
                # pow(a, number -1 ) % number  != 1
                return False
        return True
    else:
        # 확률적으로 소수판별.
        for i in range(100):
            a = random.randint(2, number - 1)
            if pow(a, number - 1, number) != 1:
                return False
        return True

```
