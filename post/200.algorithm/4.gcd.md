## 최대 공약수 - Greatest Common Divisor(Factor)

```Python
def gcd(a, b):
    while b != 0:
        result = b
        a, b = b, a % b
    return result

def test_gcd():
    number1 = 21
    number2 = 12
    assert(gcd(number1, number2) == 3)
    print("테스트통과")
```
