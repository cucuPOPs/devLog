## 회문이란?

앞에서 읽으나, 뒤에서 읽으나 똑같은 것. ex)"별똥별"

```Python
def palinderome_1(str):
    return str==str[::-1]

def palinderome_2(s):
    if len(s) < 2:
        return True
    if s[0] == s[-1]:
        return palinderome_2(s[1:-1])
    else:
        return False

```
