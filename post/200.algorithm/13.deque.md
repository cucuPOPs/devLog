양쪽 끝에서 항목의 조회, 삽입, 삭제가 가능하다.

## 구현

기존 작성하였던 Queue클래스를 상속하여 작성함

```Python
class Deque(Queue):
    def enqueue_back(self, item):
        self.items.append(item)

    def dequeue_front(self):
        value = self.items.pop(0)
        if value is not None:
            return value
        else:
            print("Deque is empty.")
```
