## 애너그램이란?

문장이나 단어에서 철자순서를 바꾸는 놀이를 말한다

ex) 'google' -> 'glegoo' (애너그램 관계인가? O)
ex) 'google' ->'goooogle' (애너그램 관계인가? X)

## 애너그램 판별 방법?

나타난 알파벳횟수가 일치하는지 체크한다

## 구현

0. 딕셔너리 자료구조를 사용하는것이 가장 좋은 해결책.
1. 2개문자열을 받는다
2. 첫번째 문자열의 문자 발생횟수를 더해 저장한다
3. 두번째 문자열의 문자 발생횟수를 빼 저장한다
4. 딕셔너리의 모든 항목값이 0이면 두 문자열은 애너그램이다.

```Python
def is_anagram_1(s1, s2):
    d = dict()
    for i in s1:
        d.setdefault(i, 0)
        d[i] += 1
    for i in s2:
        if d.get(i):
            d[i] = d[i] - 1
        else:
            return False
    for k in d.values():
        if k != 0:
            return False
    return True
```

## Counter 함수 이용하는 방법

```Python
from collections import Counter
def is_anagram2(s1, s2):
    c = Counter()
    for i in s1:
        c[i] += 1
    for i in s2:
        c[i] -= 1
    for i in c.values():
        if i != 0:
            return False
    return True
```
